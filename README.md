# 🚀 LaunchAI

**LaunchAI** is a full-stack, AI-powered career development platform that helps students and professionals accelerate their career journey. The platform leverages Generative AI to create ATS-friendly resumes, generate personalized cover letters, provide interview preparation, deliver career roadmaps, and offer industry insights tailored to each user's profile.

---

## ✨ Features

* 📄 AI-powered Resume Builder with ATS-friendly formatting
* ✉️ AI-generated Cover Letter Generator
* 🎯 AI-based Mock Interview Preparation with personalized feedback
* 🛣 Personalized Career Roadmaps based on career goals
* 📊 Real-time Industry Insights including salary trends, demand, and required skills
* 👤 Secure user authentication using NextAuth.js
* 🧭 Interactive onboarding to personalize career recommendations
* 🤖 Smart AI suggestions throughout the platform
* 📱 Fully responsive and modern user interface
* 📈 Dashboard to monitor career progress and learning journey
